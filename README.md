### USNCO exam generator agent ###

This is a skill for generating USNCO-style chemistry exams, which uses multiple subagents.

This is intended for use with the Google Antigravity agent; if you are using another agent, you may need to modify some of the formatting.

Unsure how to set up antigravity? Go to [https://antigravity.google/](https://antigravity.google/) and follow the instructions to import this as a skill.

The past_tests folder contains TeX versions of past USNCO tests. Learn more about the USNCO [here](https://www.acs.org/education/olympiad.html). Since these are not my property, they do not fall under the same license as the rest of the repo.
